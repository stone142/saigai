# 災害時職員管理システム

災害時の職員安否確認・状況管理のためのPWA（Progressive Web App）です。

## 機能

- 職員が自分の安否状況を入力
- 所属部署、現在地、参集可否などの詳細情報を管理
- 本部が全職員の状況を一覧で確認
- オフラインでも動作
- スマートフォン・タブレット・PCに対応

## 使い方

### 職員モード
1. 職員ID（半角数字）を入力
2. 氏名、所属部署、安否状況などを入力
3. 「登録する」ボタンで保存

### 本部モード
1. 管理者パスワード「saigai」を入力
2. 全職員の状況を確認

## セットアップ

1. GitHub Pagesで公開する場合：
   - リポジトリの Settings → Pages → Source で「main」ブランチを選択
   - 数分後に公開URLが表示されます

2. ローカルで使用する場合：
   - index.htmlをブラウザで開くだけで使用できます

## 注意事項

- 管理者パスワードは必ず変更してください（index.html内のADMIN_PASSWORD）
- データはブラウザのローカルストレージに保存されます
- ブラウザのデータを削除すると情報も消えます
